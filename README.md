# quark

single executable, self-hostable, realtime, golang-based, web communication application

## How to run

You can use [nix](https://nixos.org/) with [nix-flakes](https://nixos.wiki/wiki/Flakes) to run this project _effortlessly_.

Start the server

```bash
nix run github:lovedeepsingh-07/quark#server
```

Start the client

```bash
nix run github:lovedeepsingh-07/quark#client
```

You are good to go!
